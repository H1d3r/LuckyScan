# LuckyScan

>   自己手敲的一款基于Fofa api查询的信息搜集工具,  我个人感觉好用的,  暂时先不开源, 等毕业后在开源, 功能会陆续添加.

## 依赖项

基于python编写的图形化信息搜集工具, 我这边已经打包好了. 所以直接使用就好了

## 项目结构

```
UI: UI目录
LuckyScan.exe: 主程序
config.yaml: 配置文件
```
