# LuckyScan 
### 新功能正在开发中...

>   自己手敲的一款基于Fofa api查询的信息搜集工具,  我个人感觉好用的,  暂时先不开源, 等毕业后在开源, 功能会陆续添加.
>   **切勿用于非法用途, 后果自行承担, 仅限学习交流!!!!!!!**

## 依赖项

### 基于python编写的图形化信息搜集工具, 我这边已经打包好了,所以直接使用就好了, 但是需要fofa会员, 普通会员默认就是100条数据, 高级会员默认100条数据可改.

## 项目结构

```
UI: UI目录
LuckyScan.exe: 主程序
config.yaml: 配置文件
```
